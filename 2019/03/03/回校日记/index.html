回校日记

2019-03-03 21:57:29 5 分钟阅读 2128 字

开心的一天

说实话我精神状态非常差，我不是熬夜，而是通宵了。前一天晚上的八点半洗完澡出来就躺在床上玩手机，已经有些困了，发了个视频给陈思慧后就睡着了。一直睡到了 11：51 才从梦中惊醒，一看手机，几分钟前陈思慧跟我说了晚安，哦没说，因为她生气了。生气的原因是我睡着了没理她。我忙给她信息，已经没回啦，才几分钟，睡的真快。外面下很大雨，吹很大风…后来我就去 B 站看技术贴了，看到有人推荐 Linux 使用 Manjaro，兴起，起床开搞……对了，起来才发现，没关窗户，雨淋进来了！

凌晨

下载了两个版本的 Manjaro 镜像，打开 VMware，新建虚拟机…时间过得很快，因为我遇到了非常多的坑，不断 Google 希望找到问题解决。记得有一个问题，输入法无法输出中文，我找了好久估计得有一个小时，最后在看过的第一篇教程的评论区找到了问题的解决方法…几乎崩溃！然后一切都进行得很好，突然 Linux 连不上网络，又是在网上找了很多教程，改了很多文件，不停的测试，结果是，无解…此时雨已经停了，听说今天会是晴天，真期待…说回 Linux，无奈只好删除这个虚拟机，重新又装过一个…此时天慢慢的亮了起来…爷爷过来这边上厕所，吓死我，急忙关掉屏幕，不敢出声，就怕被爷爷发现我还在搞…

上午

奶奶突然走进来，问我今天陈思慧回来吗，我说不回吧，我中午就出去，和她在平南玩就好了…还问了一些其他问题就出去了，我还在弄我的电脑…不久爷爷又走进来，问我同样的问题，可见他们有多喜欢这个准孙媳妇儿？？？我还是同样的回答，期间眼睛一直盯着电脑屏幕。
要吃早餐了，在爷爷奶奶弟弟妹妹的叫声下拖拖拉拉的离开电脑前，去了卫生间洗漱，照镜子，我去！颓废啊！精神状态非常不好，洗了把脸，头还是有点晕，视线在打转！开始刷牙，扶着墙，照着镜子，你别说，还真有点帅！吃过一条粽子，一头又回到房间继续弄，这次遇到的问题是，SSR 连接上但是没办法上墙，我这强迫症，毕竟我是 Chrome 重度依赖患者嘛！
而后问题依旧没有解决，爷爷帮我拿了一大袋子的家乡特产——龙眼干，给带去学校和同学们分享。还有一个多小时，我开始收拾行李，行李真不多，很快收拾完了。十点半，我和爷爷便出发了。这次的计划是，爷爷送我到龚州中学，等陈思慧放学，她请假了，我们约好一起出去玩一个下午这样子。爷爷给她带了几条粽子和好多的大青枣，实名羡慕了哈哈哈！十一点三十多，陈思慧放学出来了，把东西放好，我们便坐上了爷爷的摩托车，一起回月亮湾放好行李，然后再出来…

下午

爷爷回家后，我们叫了车，去吃了麻辣烫。这头猪可是真厉害，点了和上次价钱一样的不同的食品！麻辣烫很好吃，陈思慧很漂亮，美食和配美女，人生巅峰了呢！下车的时候，陈思慧说要冲个话费先，我急忙说我来我来，让我表现一下，她笑笑不说话，然后说十块就好啦，我说那哪行，充三十！“那好吧！” …
吃过午餐，我们去逛了超市，玩夹娃娃机，什么嘛，不可能夹得起来的，哼！还玩了有点幼稚的赛车…哈哈哈，谁没有少女心！出来玩肯定少不了拍照啦，可是我这种直男拍照，陈思慧实在不是特别满意，我想说，拍照片是你们女孩子擅长的，各尽所用嘛！哼！哼！哼！不过我答应她，以后会给她买个相机，学一下摄影帮她拍大片，嘻嘻嘻，先听着先！
而后我们去了一家奶茶店，找了个沙发坐下来，聊天、刷视频、刷淘宝、拍照片、亲嘴…这些我们都没做！哈哈开玩笑其实都有做，陈思慧这么咸湿的一个人你说她不亲我，鬼才信！然后我就睡着了…通宵一时爽，一直通宵一直爽！怎么可能，超痛苦的，看到文章的每一个你，我都不建议你熬夜哦，因为熬夜真的真的真的对自己的手机或者电脑不好，它们也要休息！对啊！它们都要休息，何况人还是血肉之躯！好好休息！
去书店帮她的同学买了几双袜子，是的去书店买的袜子！又买了些吃的，就叫了车回龚州中学。经过一场痛苦谈不上的离别后，我坐公交回月亮湾了…

晚上

回到家就可以吃饭了，一进门，就去拿碗盛饭。吃过饭就去洗澡咯，洗过澡收拾东西，就准备出发去学校了！家人送我去了公交车站，上了公交车我倒头就睡…其实是坐着睡，真的是累啊！一觉醒来就到了高铁站，下车，打票，进站，候车，检票（差点错过，因为在发微博），上车，坐下，拿出电脑，打开电脑…一气呵成！（现在是晚上 11：10，在地铁上，笔记本就要关机了，陈思慧快回到宿舍了，先发表这么多，剩下的回到宿舍或者明天再更！）

晚上（续）

哈喽，现在是 00：56，我刚写完一篇策划案。在十二点的时候我回到宿舍了，不幸的是，我的席子发霉了，有点严重！差点就以为要再一次通宵了…好在我们团学组织部部长张瑞哲有一张席子，他的也是发霉了，他比我早到，洗了也干了！我带了一点龙眼干给他，感谢学长啊！上次说到我在动车上打开电脑，就是写那个破策划案滴，写完了还差字体以及排版，因为动车上网络不太好就没弄，这不刚弄完嘛！
我发现其实完成一项工作其实还挺有成就感的，这星期四要开一次班干会议，我一定要把上学期期末的班会总结以及班长总结给写完，其实班长总结之前有写了，但是因为那个电脑 C 盘格式化了，我的总结就不见了！！！
现在准备睡觉了，可是还想看一下书，看一下新书嘻嘻！晚安!
