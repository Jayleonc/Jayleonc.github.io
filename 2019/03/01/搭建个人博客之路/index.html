个人博客搭建之路

2019.3.1 更新

昨天，我在哔哩哔哩看到某 up 主的搭建个人博客的教程，深深被吸引，于是走上了搭建之路！

这里由是 Hexo + Github 搭建的，一开始选用的是 yilia 主题，在然后用了 next，这个主题很好看，很多人使用，网上也比较多关于这个主题的帖子，但是有一个问题就是分类和标签页没办法链接上，网上的教程也千篇一律没什么用，于是就放弃了。

在昨晚到今天的现在，我遇到了非常多的问题，有时候一句命令出错了，报错提示全是英文，翻译出来有时也难以理解，会让你折腾很久，心烦意乱！但是当它突然不报错了，你会很开心，心里想着，终于好了！有时候会觉得如果能一开始就知道问题的源头以及解决的方法，就好了！但是这就是一个学习的过程，慢慢的积累，慢慢的进步，慢慢的变成大神哈哈哈…

还有两天就开学了，回校就不会有一整天的时间去继续搭建个人博客了，但我不会放弃，加油！

2019.3.1 晚上更新

我决定日积月累，不再把全部精力都进去了，怎么说我也是后端的…前端太累了！bye.
